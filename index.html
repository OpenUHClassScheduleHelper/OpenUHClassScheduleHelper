OpenUHClassScheduleHelper

Overview

A web application that provides UH students with support for course planning and scheduling. Registering for classes can be a tremendous task so we wanted to create an application that helped with the process. Open UH Class Schedule Helper aids users in deciding what classes to take for the semester so they can achieve their educational goals.
